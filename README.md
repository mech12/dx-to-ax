# 2025-1108-ai

## 환경 설정

### Conda 환경 생성

Python 3.13 환경을 `ai`라는 이름으로 생성:

```bash
/opt/anaconda3/bin/conda create -n ai python=3.13 -y
```

### 환경 활성화

```bash
conda activate ai
```

### 환경 비활성화

```bash
conda deactivate
```

## 환경 정보

- 환경 이름: ai
- Python 버전: 3.13.9
- 환경 위치: /opt/anaconda3/envs/ai
